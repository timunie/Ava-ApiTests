      KeyboardNavigationMode Enumeration

      Defines the mode of keyboard traversal within a container when the tab or arrow keys are pressed.

      Continue0 Items in the container will be cycled through, and focus will be moved to the previous/next container after the first/last control in the container.
      Cycle1 Items in the container will be cycled through, and moving past the first or last control in the container will cause the last/first control to be focused.
      Contained2 Items in the container will be cycled through and focus will stop moving when the edge of the container is reached.
      Once3 When focus is moved into the container, the control described by the TabOnceActiveElementProperty attached property on the container will be focused. When focus moves away from this control, focus will move to the previous/next container.
      None4 The container's children will not be focused when using the tab key.
      Local5 TabIndexes are considered on local subtree only inside this container
