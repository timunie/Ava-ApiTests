      Render(DrawingContext) Draw ticks. Ticks can be draw in 8 different ways depends on Placement property and IsDirectionReversed property. This function also draw selection-tick(s) if IsSelectionRangeEnabled is 'true' and SelectionStart and SelectionEnd are valid. The primary ticks (for Minimum and Maximum value) height will be 100% of TickBar's render size (use Width or Height depends on Placement property). The secondary ticks (all other ticks, including selection-tics) height will be 75% of TickBar's render size. Brush that use to fill ticks is specified by Fill property.
      (Overrides Visual.Render(DrawingContext))
